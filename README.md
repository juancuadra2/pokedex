## Installation

1. Clone the repository
    ```bash
    $ git clone ...
    ```

2. Install dependencies
    ```bash
    $ npm install
    ```

3. Install nest CLI
    ```bash
    $ npm install -g @nestjs/cli
    ```
   
4. run database
    ```bash
    $ docker-compose run -d
    ```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
